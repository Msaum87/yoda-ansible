yoda-ansible
============

REQUIREMENTS
------------
    * [Ansible](https://docs.ansible.com/ansible/intro_installation.html)
    * [VirtualBox](https://www.virtualbox.org/manual/ch02.html)
    * [Vagrant](https://www.vagrantup.com/docs/installation/)

USAGE
-----
```bash
vagrant up
ansible-playbook playbook.yml
```
